# YouTube Analytics Dashboards

A set of Streamlit dashboards for exploring YouTube channel performance, growth, and sponsorship opportunities.

## Quick start
1. Install dependencies (example):
   ```bash
   pip install streamlit pandas numpy plotly
   ```
2. Run the primary dashboard (accessibility-focused):
   ```bash
   streamlit run final_dashboard.py
   ```

## Data
- Default CSVs: `cleaned_youtube_data_3.csv`, `cleaned_youtube_data_new_1.csv`
- You can upload your own CSV from the sidebar upload control.

## Notes
- Accessibility: `final_dashboard.py` includes dark-mode defaults and accessibility toggles.
- If you see missing packages, install them with `pip install <name>` and rerun the dashboard.
